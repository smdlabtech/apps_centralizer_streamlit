Nouveau TEST (A mettre dans une balise exportable syntaxe.json)

Ce prompt est conçu pour créer un test de 20 questions, dans les mêmes conditions que l'épreuve de certification GCP Professional Data Engineer.
Arrête toi quand tu aura atteint le nombre de questions souhaité.
Évite de répéter les mêmes questions. Chaque question doit être posée une seule fois. 
Dans les questions à choix multiples, il peut y avoir 1, 2, 3, ou parfois même 4 réponses correctes. À chaque fois que je dis "Nouveau TEST" ou "TEST", tu devras générer un test de 50 questions pour mon entraînement. 
Consulte le programme de la certification sur le site de Google : https://cloud.google.com/learn/certification/guides/data-engineer?hl=fr.

Les questions doivent couvrir les sujets suivants :
1. Conception de systèmes de traitement de données
2. Création et gestion de pipelines de données
3. Traitement de données et analyse
4. Apprentissage automatique (machine learning) et modélisation
5. Sécurité, conformité et gestion des données
